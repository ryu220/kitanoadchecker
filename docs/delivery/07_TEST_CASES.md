```bash

http://localhost:3000 でテキスト入力 → 商品選択 → チェック開始

# APIでテスト

  -H "Content-Type: application/json" \
  -d @test-data.json
```

## 1. ギネス期間検証（HA商品）

### ❌ NG例: 開始年1年早い
```
売上世界一※1を達成したヒアロディープパッチ。
※1：ディープパッチシリーズとして売上世界一（TFCO株式会社のグローバル調査、美容用マイクロニードルスキンパッチにおける最大のブランド、2019年3月～2024年2月の5年間）
```
**違反**: 開始年が2019年（正しくは2020年）

### ✅ OK例: 正しい期間
```
ギネス世界記録™認定※1 売上世界一※1。
※1：ディープパッチシリーズとして売上世界一（TFCO株式会社のグローバル調査、美容用マイクロニードルスキンパッチにおける最大のブランド、2020年～2024年の5年間）でギネス世界記録™認定
```

## 2. ランキング表現（Issue #36）

### ❌ NG例: エビデンスなし
```
Amazon・楽天で1位を獲得した人気商品です。
```
**違反**: 「1位」に調査機関・期間のエビデンス注釈なし（景表法違反）

### ✅ OK例: エビデンスあり
```
Amazon・楽天で1位※を獲得。
※2024年1月Amazon・楽天美容部門ランキング調査
```

---

## 3. 浸透表現（HA商品）

### ❌ NG例: 範囲不明確
```
ヒアルロン酸が肌の奥深くまで浸透します。
```
**違反**: 浸透範囲の注釈なし（薬機法違反）

### ✅ OK例: 角質層まで明示
```
ヒアルロン酸が角層※まで浸透します。※角層まで
```

---

## 4. クマ表現（HA商品）

### ❌ NG例: 注釈なし
```
クマに悩む方におすすめです。
```
**違反**: 「クマ」単体は化粧品効能56項目外、注釈必須

### ✅ OK例: 注釈あり
```
目元のクマ※に悩む方におすすめです。
※乾燥や古い角質によるくすみ、ハリ不足
```

---

## 5. 殺菌表現（SH商品）

### ❌ NG例: 作用機序不明
```
爪をきれいにする殺菌ジェル。
```
**違反**: 「殺菌」は作用機序の注釈必須

### ✅ OK例: 注釈あり
```
爪をきれいにする殺菌※ジェル。
※殺菌は消毒の作用機序として
```

## 6. NO.1表現

### ❌ NG例: エビデンスなし
```
売上NO.1のヒアルロン酸パッチ
```
**違反**: 「NO.1」に調査機関・期間のエビデンス必須

### ✅ OK例: エビデンスあり
```
売上NO.1※のヒアルロン酸パッチ
※2024年1月 ○○社調べ、ヒアルロン酸パッチ市場
```

## 7. 保証表現

### ❌ NG例: 条件不明
```
効果がなければ全額返金保証！
```
**違反**: 保証条件の明示なし（景表法違反）

### ✅ OK例: 条件明示
```
全額返金保証※
※初回購入30日以内、条件詳細はサイト参照
```

---

## 8. 体験談

### ❌ NG例: 断定的表現
```
使って1週間でシワが消えました！（50代女性）
```
**違反**: 効果を断定、「個人の感想です」注釈なし

### ✅ OK例: 注釈あり
```
使って1週間で肌にハリを感じました※（50代女性）
※個人の感想です。効果を保証するものではありません。
```

---

## 9. ステマ規制

### ❌ NG例: PR表記なし
```
【SNS投稿風】このパッチすごくいい！友達にもオススメしたい♪
```
**違反**: 広告であることが不明（ステマ規制違反）

### ✅ OK例: PR表記あり
```
【PR】このパッチすごくいい！友達にもオススメしたい♪
```

---

## 10. 若返り表現

### ❌ NG例: 化粧品効能超え
```
肌が若返り、10歳若く見えます。
```
**違反**: 「若返り」は化粧品で訴求不可（薬機法違反）

### ✅ OK例: 許容表現
```
若々しい印象※の肌へ。
※メーキャップ効果による
```
