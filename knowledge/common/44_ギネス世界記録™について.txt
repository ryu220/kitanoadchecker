## ギネス世界記録™について

### 正式な認定期間（最重要）

**【絶対厳守】正しい期間: 2020年～2024年の5年間**

『ヒアロディープパッチ』『ミケンディープパッチ』『オデコディープパッチ』『チークポアパッチ』は、*※2020年～2024年の5年連続**で「美容用マイクロニードルパッチブランド」として売上世界一であることが、ギネス世界記録™に認定されています。

**認定内容の詳細:**
- **開始年:** 2020年（この年から開始）
- **終了年:** 2024年（この年まで継続）
- **期間:** 5年連続（2020, 2021, 2022, 2023, 2024）
- **調査機関:** TFCO株式会社のグローバル調査
- **調査対象:** 美容用マイクロニードルスキンパッチにおける最大のブランド
- **認定内容:** 売上世界一

### 期間検証の重要性

**【景表法上の重大リスク】**

期間表記を誤ると、以下の法的リスクが発生します：
- 優良誤認（実際より長い期間を訴求）
- 不実証広告（証明できない期間の主張）
- 景品表示法違反（措置命令・課徴金の対象）

**このため、期間は1年たりとも間違えてはいけません。**

### よくある誤り（絶対NG）

#### ❌ NG例1: 開始年が早すぎる

**誤:** 2019年3月～2025年2月
**理由:** 開始年が2019年となっており、正しい2020年より1年早い
**違反内容:** 実際より長い期間を訴求（優良誤認）

**誤:** 2019年～2024年
**理由:** 開始年が2019年となっており、正しい2020年より1年早い
**違反内容:** 6年間と誤認される可能性

#### ❌ NG例2: 終了年が遅すぎる

**誤:** 2020年～2025年
**理由:** 終了年が2025年となっており、正しい2024年より1年遅い
**違反内容:** 実際より長い期間を訴求（優良誤認）

**誤:** 2020年～2025年2月
**理由:** 終了年月が2025年2月となっており、認定期間を超過
**違反内容:** 未認定の期間を含めている

#### ❌ NG例3: 期間の長さが間違っている

**誤:** 2020年～2024年の6年間
**理由:** 期間が「6年間」となっており、正しくは「5年間」
**違反内容:** 事実と異なる期間の主張

**誤:** 2020年～2025年の5年間
**理由:** 終了年が2025年で誤り（正しくは2024年）
**違反内容:** 期間の長さは合っているが年が間違っている

#### ❌ NG例4: 月単位の指定による誤り

**誤:** 2019年3月～2024年2月（5年間）
**理由:** 開始月が2019年3月で、2020年より前
**違反内容:** 開始時期が1年早い

### 検証ルール（必須チェック項目）

広告文で「売上世界一」「ギネス世界記録™」を訴求する際は、以下を厳密にチェックしてください：

#### ✅ チェック項目1: 開始年が2020年であること
- 2019年 → **NG**
- 2020年 → **OK**
- 2021年 → **NG**（開始年として）

#### ✅ チェック項目2: 終了年が2024年であること
- 2023年 → **NG**（終了年として）
- 2024年 → **OK**
- 2025年 → **NG**

#### ✅ チェック項目3: 期間の長さ（明示されている場合のみ）
- 「N年間」「N年連続」と明記されている場合のみチェック
  - 4年間 → **NG**
  - 5年間/5年連続 → **OK**
  - 6年間 → **NG**
- 年の範囲のみ（「2020年～2024年」など）の場合は、開始年と終了年が正しければ**OK**

#### ✅ チェック項目4: 調査機関名が正確であること
- 必須記載: **TFCO株式会社のグローバル調査**
- 省略や変更は不可

#### ✅ チェック項目5: 調査対象範囲が正確であること
- 必須記載: **美容用マイクロニードルスキンパッチにおける最大のブランド**
- 省略や言い換えは不可

### 正しい表記例（OK例）

#### ✅ OK例1: 基本形
```
売上世界一※1
※1：ディープパッチシリーズとして売上世界一（TFCO株式会社のグローバル調査、美容用マイクロニードルスキンパッチにおける最大のブランド、2020年～2024年）
```

#### ✅ OK例2: 期間の長さを明示（「5年連続」）
```
売上世界一※1
※1：ディープパッチシリーズとして売上世界一（TFCO株式会社のグローバル調査、美容用マイクロニードルスキンパッチにおける最大のブランド、2020年～2024年の5年連続）
```

#### ✅ OK例3: 期間の長さを明示（「5年間」）
```
売上世界一※1
※1：ディープパッチシリーズとして売上世界一（TFCO株式会社のグローバル調査、美容用マイクロニードルスキンパッチにおける最大のブランド、2020年～2024年の5年間）
```

**注:** 「5年間」と「5年連続」は同じ意味で、どちらも正しい表現です。

#### ✅ OK例4: ギネス世界記録™と明記
```
ギネス世界記録™認定 売上世界一※1
※1：ディープパッチシリーズとして売上世界一。TFCO株式会社のグローバル調査、美容用マイクロニードルスキンパッチにおける最大のブランド、2020年～2024年の5年連続でギネス世界記録™に認定
```

### AI評価への指示

**【AI評価担当者への重要指示】**

セグメント内に以下のキーワードが含まれる場合、必ず期間検証を実行してください：
- 「売上世界一」
- 「世界一」
- 「ギネス」
- 「No.1」

**検証手順:**
1. 注釈から期間表記を抽出（例: 2019年3月～2025年2月）
2. 開始年が2020年であるか確認 → 違う場合は**違反**
3. 終了年が2024年であるか確認 → 違う場合は**違反**
4. 「N年間」「N年連続」と明記されている場合のみ、期間の長さをチェック
   - 「6年間」「4年間」など、5年以外が明記されている → **違反**
   - 「2020年～2024年」のように年範囲のみの場合 → 開始年・終了年が正しければ**OK**

**違反時のメッセージ形式:**
```
【景表法違反・優良誤認】期間表記が誤っています。

記載期間: 「[抽出した期間]」
正しい期間: 「2020年～2024年の5年間」
誤りの内容: [具体的な差異を記載]

修正方法: 注釈の期間を「2020年～2024年」または「2020年～2024年の5年連続」に修正してください。
```

**重要:** この検証は見逃し厳禁です。期間が1年でもずれている場合は必ず違反として検知してください。
